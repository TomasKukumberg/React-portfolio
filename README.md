## Run the project

### 1. Clone the repo
```bash
git clone https://github.com/TomasKukumberg/React-portfolio.git
```
### 2. Change into directory
```bash
cd React-portfolio
```
### 3. Install dependencies
```bash
npm install
```
### 4. Run the project locally
```bash
npm run dev
```
